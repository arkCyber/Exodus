# Hermes 智能体应用案例

本目录包含 Hermes 智能体的实际应用案例，展示如何在 Exodus 浏览器中使用智能体自动化各种任务。

## 案例列表

1. **网页自动化** (`web_automation/`) - 自动化网页导航和交互
2. **数据提取** (`data_extraction/`) - 从网页提取结构化数据
3. **表单填充** (`form_filling/`) - 自动填充和提交表单
4. **策略链** (`strategy_chain/`) - 多步骤复杂任务编排

## 快速开始

每个案例目录包含：
- `frontend.ts` - 前端 TypeScript 调用示例
- `config.json` - 后端配置示例
- `README.md` - 详细使用说明

- ✅ 可直接复制使用
