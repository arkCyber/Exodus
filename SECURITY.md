# Security Policy

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a vulnerability

Please **do not** open public GitHub issues for security-sensitive reports.

1. Email or DM the maintainers privately (add your contact in the repository settings after publish).
2. Include steps to reproduce, impact, and affected components (`src-tauri`, `exodus-core`, etc.).
3. Allow reasonable time for a fix before public disclosure.

We will acknowledge receipt and work on remediation as soon as possible.
