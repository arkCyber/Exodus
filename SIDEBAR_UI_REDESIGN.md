# 侧边栏 UI 重新设计 - Firefox 风格

## 设计目标

对齐 Firefox 浏览器的侧边栏设计，提供更现代、更直观的用户体验。

## Firefox 侧边栏特点分析

### 布局结构
- **双栏布局**: 左侧窄图标栏 (48px) + 右侧宽内容区 (320px)
- **可折叠**: 图标栏可独立折叠，节省空间
- **清晰分隔**: 图标栏和内容区有明显的视觉分隔

### 视觉风格
- **简洁现代**: 扁平化设计，适度阴影
- **图标优先**: 使用清晰的图标表示功能
- **悬停反馈**: 明确的悬停状态和交互反馈
- **主题一致**: 与浏览器整体主题保持一致

### 交互体验
- **快速切换**: 点击图标快速切换面板
- **键盘导航**: 支持键盘快捷键
- **拖拽调整**: 可拖拽调整宽度（可选）
- **动画过渡**: 平滑的展开/收起动画

## Exodus 当前侧边栏问题

1. **布局单一**: 单一宽栏布局，没有图标栏
2. **视觉层次**: 标签栏和内容区区分不明显
3. **交互反馈**: 悬停状态不够明显
4. **空间利用**: 固定宽度，无法折叠

## 新设计方案

### 布局结构

```
┌─────────────────────────────────────┐
│  ┌──┐  ┌─────────────────────────┐  │
│  │AI│  │  Panel Title           │  │
│  ├──┤  ├─────────────────────────┤  │
│  │🕐│  │                         │  │
│  ├──┤  │  Panel Content          │  │
│  │★ │  │                         │  │
│  ├──┤  │                         │  │
│  │👥│  │                         │  │
│  ├──┤  │                         │  │
│  │📦│  │                         │  │
│  └──┘  └─────────────────────────┘  │
└─────────────────────────────────────┘
  48px         320px
```

### 功能面板

1. **AI Chat** - AI 对话和摘要
2. **History** - 浏览历史和索引记忆
3. **Bookmarks** - 书签管理
4. **P2P** - P2P CDN 和群聊
5. **Pocket** - Pocket 集成

### 视觉改进

- **图标栏**: 48px 宽，深色背景，图标居中
- **分隔线**: 1px 边框分隔图标栏和内容区
- **悬停效果**: 图标悬停时显示工具提示和背景高亮
- **活动状态**: 当前面板图标高亮显示
- **折叠按钮**: 底部折叠/展开按钮

### 交互改进

- **快速切换**: 点击图标立即切换面板
- **键盘快捷键**: Cmd/Ctrl + Shift + S 切换侧边栏
- **拖拽调整**: 拖拽边缘调整内容区宽度
- **动画过渡**: 200ms 平滑过渡动画

## 实现计划

### 阶段 1: 基础结构
- 创建新的侧边栏组件 `FirefoxStyleSidebar.svelte`
- 实现双栏布局
- 添加图标栏和内容区

### 阶段 2: 视觉样式
- 实现 Firefox 风格的视觉设计
- 添加悬停效果和动画
- 实现主题适配

### 阶段 3: 交互功能
- 实现面板切换
- 添加折叠/展开功能
- 实现键盘快捷键

### 阶段 4: 集成
- 替换现有侧边栏组件
- 更新浏览器主界面
- 测试和调试

## 技术栈

- **Svelte 5**: 使用 Svelte 5 的 runes API
- **TailwindCSS**: 用于样式（可选）
- **Lucide Icons**: 图标库
- **CSS Transitions**: 动画效果
