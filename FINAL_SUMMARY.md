### 标签页休眠流程

```
1. 用户打开标签页
   ↓
2. 注册到 TabSleepManager
   ↓
3. 监控活跃状态
   ↓
4. 超过阈值 (5分钟)
   ↓
5. 检查排除条件
   - 是否固定？
   - 是否播放媒体？
   ↓
6. 标记为休眠
   ↓
7. 释放内存
   ↓
8. 用户切换回标签页
   ↓
9. 唤醒标签页
   ↓
10. 恢复内容
```

## 📚 API 文档

### 标签页休眠 API

```typescript
// 注册标签页
await invoke('tab_sleep_register', {
  tabId: 'tab-123',
  label: 'webview-1',
  url: 'https://example.com',
  title: 'Example',
  isPinned: false
});

// 标记为活跃
await invoke('tab_sleep_mark_active', {
  tabId: 'tab-123'
});

// 更新媒体状态
await invoke('tab_sleep_update_media', {
  tabId: 'tab-123',
  isPlayingAudio: true,
  isPlayingVideo: false
});

// 获取统计
const stats = await invoke('tab_sleep_get_stats');
console.log(`节省内存: ${stats.saved_memory_mb} MB`);

// 更新配置
await invoke('tab_sleep_update_config', {
  config: {
    enabled: true,
    inactive_threshold_secs: 300,
    max_active_tabs: 10,
    exclude_media: true,
    exclude_pinned: true
  }
});
```

### 指标收集 API

```typescript
// 记录计数器
await invoke('metrics_counter', {
  name: 'http_requests_total',
  value: 1.0,
  labels: { method: 'GET', status: '200' }
});

// 记录仪表盘
await invoke('metrics_gauge', {
  name: 'memory_usage_bytes',
  value: 1024 * 1024 * 512,
  labels: { type: 'heap' }
});

// 记录直方图
await invoke('metrics_histogram', {
  name: 'request_duration_seconds',
  value: 0.123
});

// 获取所有指标
const metrics = await invoke('metrics_get_all');

// 导出 Prometheus 格式
const prometheus = await invoke('metrics_export_prometheus');
```
